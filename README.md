## О проекте
Приложение для мониторинга датчиков, контролирующих праметры микроклимата в помещениях

## Пользователи
В системе 2 предустановленных пользователя:
### -admin (пароль admin) c ролью Administrator
### -user (пароль user) c ролью Viewer
